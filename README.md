## Windows
用文本编辑器（如 Notepad++）打开文件：
C:\Windows\System32\drivers\etc\hosts
将 hosts 全部内容复制到上述文件内并保存.
注意：如果遇到无法保存，请右键文件hosts并找到“属性” -> “安全”，然后选择你登陆的用户名，最后点击编辑，勾选“写入”即可。

## Linux 和 Mac OS
将 hosts 全部内容复制到/etc/hosts中并保存。
